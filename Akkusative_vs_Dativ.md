### Akkusative → Direct Object
English: WHOM? / WHAT?

German: Wen? / Was?

Example:
Ich sehe den Hund.
(Whom do I see? → den Hund → Akkusativ)

### Dativ kya hota hai? → Indirect Object
English: TO WHOM? / FOR WHOM?

German: Wem?

Example:
Ich gebe dem Mann den Ball.
(To whom do I give the ball? → dem Mann → Dativ)
(What do I give? → den Ball → Akkusativ)

### Example
Sentence:

Ich gebe dem Kind das Buch.

das Buch → What is giving? → direct → Akkusativ

dem Kind → To whom am I giving? → receiver → Dativ

| Case       | Question (German) | Question (English)     | Use For                       | Article Changes (Short)                  |
|------------|--------------------|--------------------------|--------------------------------|------------------------------------------|
| Akkusativ  | Wen? / Was?        | Whom? / What?            | Direct object (direct hit)     | der → den (only masculine)              |
| Dativ      | Wem?               | To whom? / For whom?     | Indirect object (receiver)     | der → dem, die → der, das → dem, die(pl) → den + n |

| Sentence Role       | Case        | Example                    |
|---------------------|-------------|-----------------------------|
| Subject             | Nominativ   | Der Mann isst.              |
| Direct Object       | Akkusativ   | Ich sehe den Hund.          |
| Indirect Object     | Dativ       | Ich gebe dem Kind ein Buch. |

| Case      | Masculine       | Feminine      | Neuter        | Plural            |
|-----------|------------------|---------------|---------------|--------------------|
| Akkusativ | den / einen      | die / eine    | das / ein     | die / —            |
| Dativ     | dem / einem      | der / einer   | dem / einem   | den+n / keinen+n   |
